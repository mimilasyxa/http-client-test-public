## Первоначальный запуск приложения

1. **Изменение файла `/docker/.env.example`**  
   Обновите следующие переменные в соответствии с вашим окружением:
    - `APP_NAME` — укажите название вашего приложения.
    - `DB_HOST` — задайте имя контейнера с базой данных.
    - `REDIS_HOST` — задайте имя контейнера с Redis.
    - При необходимости добавьте или измените другие переменные.
2. **Изменение файла `/app/.env.example`**  
   Обновите следующие переменные в соответствии с вашим окружением:
    - `APP_NAME` — укажите название вашего приложения.
    - `DB_HOST` — задайте имя контейнера с базой данных.
    - `REDIS_HOST` — задайте имя контейнера с Redis.
    - `TEST_API_URL` - ссылка на тестовое API
    - `TEST_API_KEY` - ключ для тестового API
